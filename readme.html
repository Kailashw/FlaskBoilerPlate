Follow these instructions before you create any new project.

  • upgrade pip using 'py -m pip install --upgrade pip'
    • Note : use 'py -m pip --version' for checking current pip version.
  • Install virtualenv using 'py -m pip install --user virtualenv'
  • create a virtual env for this project using 'py -m virtualenv venv' command
  • activate the virtual env 'venv' by using actiavting scripts under it's scripts folder.
  • type command 'deactivate' anytime to come out of current virtual environmnt.
  • you can install requiremtns from requirements.txt file by using 'pip install -r requirements.txt'
  • you can anytime write to the same file with new requirements by using 'pip freeze > requirements.txt' command.
